
1. **`<!DOCTYPE html>`**: HTML 문서의 형식을 정의
   ```html
   <!DOCTYPE html>
   ```

2. **`<html>`**: HTML 문서의 시작을 나타냄
   ```html
   <html lang="en">
   ```

3. **`<head>`**: 문서의 메타데이터를 포함하는 부분
   ```html
   <head>
       <meta charset="UTF-8">
       <meta name="viewport" content="width=device-width, initial-scale=1.0">
       <title>문서 제목</title>
   </head>
   ```

4. **`<body>`**: 실제 콘텐츠가 위치하는 부분
   ```html
   <body>
       <h1>안녕하세요, 웹 페이지!</h1>
       <p>이것은 간단한 문장입니다.</p>
   </body>
   ```

5. **`<h1>` to `<h6>`**: 제목을 나타내는 태그로, `<h1>`이 가장 크고 `<h6>`이 가장 작음
   ```html
   <h1>제목 1</h1>
   <h2>제목 2</h2>
   ```

6. **`<p>`**: 문단을 나타내는 태그
   ```html
   <p>이것은 단락입니다.</p>
   ```

7. **`<a>`**: 하이퍼링크를 만드는 태그
   ```html
   <a href="https://www.example.com">바로 가기</a>
   ```

8. **`<img>`**: 이미지를 삽입하는 태그
   ```html
   <img src="image.jpg" alt="이미지 설명">
   ```

9. **`<ul>` and `<li>`**: 순서가 없는 목록을 나타내는 태그
   ```html
   <ul>
       <li>항목 1</li>
       <li>항목 2</li>
   </ul>
   ```

10. **`<table>`**, **`<tr>`**, **`<td>`**: 표를 만드는 태그
    ```html
    <table border="1">
        <tr>
            <td>셀 1</td>
            <td>셀 2</td>
        </tr>
    </table>
    ```
